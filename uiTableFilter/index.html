Filtragem de uma tabela com o uiTableFilter

O uiTableFilter é um plugin Jquery que serve para filtrar as linhas de uma tabela segundo um critério de busca, como mostrado na figura abaixo:

Seu uso é bastante simples e é mostrado neste código de exemplo:

<script type="text/javascript">
	$(document).ready(function(){
		var $tabela = $('table#resultados');
		$("#busca").keyup(function() {
			$.uiTableFilter($tabela, this.value);
		});
	});	      
</script>
